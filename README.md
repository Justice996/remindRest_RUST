# 休息提醒助手 (Rest Reminder)

一个用 Rust 和 egui 编写的智能番茄工作法休息提醒工具，帮助你管理工作时间，提醒定时休息，保护健康。

## ✨ 特性

- 🍅 **番茄工作法支持**: 可自定义工作和休息时长
- 🖥️ **智能全屏提醒**: 工作时间结束后自动全屏显示，确保你不会错过休息
- 🌟 **蒙层休息模式**: 半透明全屏蒙层，既醒目又不完全遮挡视线
- 🎨 **动态表情动画**: 休息时有可爱的表情符号掉落动画
- 📱 **用户友好控制**: 支持暂停、跳过、提前结束休息等操作
- ⚡ **自动最小化**: 休息结束后程序自动最小化，让你决定何时开始新工作
- 🎯 **智能状态管理**: 自动在工作和休息模式间切换，支持手动控制

## 🚀 快速开始

### 环境要求

- Rust 1.70+
- Windows 操作系统（推荐）

### 安装运行

1. **克隆项目**
```bash
git clone <repository-url>
cd world_hello
```

2. **编译运行**
```bash
cargo run
```

3. **或者直接编译**
```bash
cargo build --release
```
编译后的可执行文件位于 `target/release/world_hello.exe`

## 📖 使用指南

### 基本工作流程

1. **启动程序**: 运行程序后会显示控制面板
2. **开始工作**: 点击"开始工作"按钮开始计时
3. **工作时间**: 程序以正常窗口显示，倒计时剩余工作时间
4. **自动休息**: 工作时间结束后自动进入全屏休息模式
5. **休息时间**: 显示半透明蒙层和大字体倒计时
6. **自动最小化**: 休息结束后程序自动最小化到任务栏
7. **重新开始**: 从任务栏恢复程序，手动开始新的工作周期

### 界面模式

#### 工作模式
- 正常窗口显示
- 完整的控制面板
- 工作时长和休息时长设置
- 倒计时显示和控制按钮

#### 休息模式
- 全屏半透明蒙层
- 大字体休息倒计时
- 表情符号掉落动画
- "提前结束休息"按钮

### 控制选项

- **开始工作**: 开始新的工作周期
- **暂停**: 暂停当前计时器
- **跳过**: 立即切换到下一个阶段
- **提前结束休息**: 最小化程序并暂停计时器

## ⚙️ 配置

### 默认设置
- 工作时长: 25 分钟
- 休息时长: 5 分钟

### 自定义设置
在控制面板的设置区域可以：
- 修改工作时长（分钟）
- 修改休息时长（分钟）
- 点击"确定"按钮应用新设置

## 🛠️ 技术实现

### 核心技术栈
- **语言**: Rust (Edition 2024)
- **GUI框架**: egui + eframe 0.28
- **动画**: 自定义表情符号掉落系统
- **窗口管理**: Viewport 命令和全屏控制

### 关键功能
- **状态管理**: 工作/休息/暂停三状态自动切换
- **定时器系统**: 高精度时间计算和倒计时
- **动画引擎**: 60fps 表情符号掉落效果
- **UI适配**: 自动根据状态切换界面样式
